# Security Policy

## Supported Versions

Security vulnerabilities are actively monitored and fixed only for the current major release.

If you are using an older version, please upgrade to the latest stable release to ensure your application remains secure.

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

If a potential security vulnerability is discovered in this library, please **DO NOT open a public issue** on GitHub. Publicly disclosing a vulnerability makes it easier for attackers to exploit it before a patch is available.

Instead, please report it responsibly by sending an email directly to the maintainer: **dimogrudev98@gmail.com**
